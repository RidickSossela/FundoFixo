# Fundo Fixo
Sistema de controle de gastos.
    - Login.
    - Emissão de relatorios por periodo em formato PDF.
    - Gerenciamento de itens.
    - Pesonalização de categorias de gastos.
    - Preenchimento automatico do periodo do relatorio, valor total de gastos.
    - Converte valor do fundo fixo em valor por extenso. 

Desenvolvido com frameworks Laravel e Vue JS.
